Percentage difference calculator to find percent difference between two numbers. The calculator is used to compare two positive values.

[https://www.calculator.io/percentage-difference-calculator/](https://www.calculator.io/percentage-difference-calculator/)

== Usage ==

Add the Percentage Difference Calculator shortcode to your page, post or sidebar:

`[ci_percentage_difference_calculator]`

Add the following code to your template where you would like the Percentage Difference Calculator to appear:

`<?php display_ci_percentage_difference_calculator(); ?>`
